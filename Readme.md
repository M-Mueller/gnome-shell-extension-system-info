Installation
------------
Copy the **system-info@<span>M-Mueller.github</span>.com** folder to ~/.local/share/gnome-shell/extensions and activate it through [Gnome Tweak Tool](http://wiki.gnome.org/GnomeTweakTool)

### Compatible with Gnome Shell:
- 3.18
- 3.20
